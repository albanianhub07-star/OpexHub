# OpexHub
Opex Hub

## Usage

1. Place `OpexHub.lua` into your Roblox executor's script folder or load it manually.
2. Execute the script in the game.
3. Use the Rayfield UI to enable:
   - `Auto Harvest`
   - `Auto Water`
   - `Harvest Once`
   - `Water Once`
   - `⚠️ Duplicate Pet / Add to Inventory ⚠️`

> Note: Pet duplication is experimental and depends on the game. The script also tries to clone pets into inventory and call garden/inventory remotes, but server-side validation may still block it.
